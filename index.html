Cine suntem?

O FAMILIE PASIONATA

Ce livram?

SIROPURI NATURALE

Livram siropuri naturale, realizate din flori si fructe atent culese, din zonele nepoluate de deal si munte ale Transilvaniei.

Ce oferim?

SIGURANTA CALITATII

Aspectul esential de care tinem cont atunci cand culegem / achizitionam materia prima pentru siropurile pe care le producem este calitatea fiecarui ingredient folosit.

De ce noi?

SIGURANTA SI CALITATE

Modul nostru de gandire este foarte simplu:

2. gusti,
3. iti place → pastrezi
4. nu iti place → pastrezi si primesti banii inapoi
